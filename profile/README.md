# Skill HQ

Open-source CLI tools and agent skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [OpenClaw](https://github.com/nichochar/open-claw), [Codex](https://github.com/openai/codex), and other AI coding agents.

## 🛠️ Tools

| Tool | Description | Install |
|------|-------------|---------|
| [**telegram**](https://github.com/skillhq/telegram) | Fast Telegram CLI — read, search, and send messages | `npx skills add https://github.com/skillhq/telegram` |
| [**onchain**](https://github.com/skillhq/onchain) | Crypto portfolio tracking, market data, and CEX history | `npx skills add https://github.com/skillhq/onchain` |
| [**hubspot**](https://github.com/skillhq/hubspot) | HubSpot CRM — manage contacts, companies, deals, and engagements | `npx skills add https://github.com/skillhq/hubspot` |
| [**concierge**](https://github.com/skillhq/concierge) | Find contact details for accommodation listings | `npx skills add https://github.com/skillhq/concierge` |
| [**linkedin**](https://github.com/skillhq/linkedin) | LinkedIn integration via browser automation | `npx skills add https://github.com/skillhq/linkedin` |

## 🤖 Agent Skills

| Skill | Description | Install |

## 📦 Getting Started

Install any skill with a single command:

```bash
npx skills add https://github.com/skillhq/<skill-name>
```

See individual repos for configuration and usage details.
