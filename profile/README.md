# Skill HQ

Open-source CLI tools and agent skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## 🛠️ Tools

| Tool | Description | Install |
|------|-------------|---------|
| [**telegram**](https://github.com/skillhq/telegram) | Fast Telegram CLI — read, search, and send messages | `npm i -g @skillhq/telegram` |
| [**onchain**](https://github.com/skillhq/onchain) | Crypto portfolio tracking, market data, and CEX history | `npm i -g @skillhq/onchain` |
| [**hubspot**](https://github.com/skillhq/hubspot) | HubSpot CRM — manage contacts, companies, deals, and engagements | `npm i -g @skillhq/hubspot` |
| [**concierge**](https://github.com/skillhq/concierge) | Find contact details for accommodation listings | `npm i -g @skillhq/concierge` |
| [**linkedin**](https://github.com/skillhq/linkedin) | LinkedIn integration via browser automation | `npm i -g @skillhq/linkedin` |

## 🤖 Agent Skills

Skills that extend Claude Code with new capabilities — no CLI installation required.

| Skill | Description |
|-------|-------------|
| [**x-research**](https://github.com/skillhq/x-research) | X/Twitter research agent powered by the bird CLI |
| [**flight-search**](https://github.com/skillhq/flight-search) | Search Google Flights for prices and schedules |
| [**hotel-search**](https://github.com/skillhq/hotel-search) | Search Google Hotels for prices, ratings, and availability |

## 📦 Getting Started

Each tool is a standalone CLI that Claude Code can use as an agent skill. Install via npm, add the corresponding `SKILL.md` to your Claude Code config, and you're ready to go.

See individual repos for setup instructions.
